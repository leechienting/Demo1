📌 第一步：取得 API Key（2 分鐘）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 用 Chrome 打開：https://console.anthropic.com/
2. 點擊右上角「Sign In」或「Sign Up」註冊/登入
3. 登入後，點擊左側選單的「API Keys」
4. 點擊「Create Key」建立新的 API Key
5. 複製顯示的 API Key（sk-ant-開頭的長串文字）
   ⚠️ 注意：只會顯示一次，請妥善保存！

💰 費用：新用戶通常有免費額度，之後按使用量計費
         （一般使用約 $0.001-0.003 美元/次，非常便宜）


📌 第二步：安裝擴充功能（2 分鐘）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 打開 Chrome 瀏覽器

2. 在網址列輸入並按 Enter：
   chrome://extensions/

3. 在右上角開啟「開發人員模式」開關
   （Developer mode）

4. 點擊左上角「載入未封裝項目」
   （Load unpacked）

5. 在彈出的視窗中，選擇這個資料夾：
   /Users/chientinglee/Documents/Project/Demo1/language-learning-extension

6. 點擊「選擇」，完成！

7. 你會在 Chrome 右上角看到 🎓 圖示


📌 第三步：設定 API Key（1 分鐘）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 點擊 Chrome 右上角的 🎓 圖示

2. 切換到「⚙️ 設定」頁籤

3. 在「Anthropic API Key」欄位貼上你的 API Key

4. 選擇解釋語言（預設：繁體中文）

5. 點擊「💾 儲存設定」

6. 看到「設定已儲存！」就完成了


📌 開始使用！
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 打開任何網頁（例如：維基百科、新聞網站、英文文章）

2. 用滑鼠選取你想了解的文字

3. 等待 1-2 秒，會出現 AI 解釋視窗

4. 點擊「💾 加入單字表」可以儲存單字

5. 點擊 🎓 圖示可以查看你的單字表


📌 常見問題
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q: 選取文字後沒反應？
A: 重新載入網頁（按 F5），並確認已設定 API Key

Q: 顯示「請先輸入 API Key」？
A: 點擊 🎓 圖示 → 設定 → 貼上 API Key → 儲存

Q: 想換成英文解釋？
A: 點擊 🎓 圖示 → 設定 → 解釋語言選「English」→ 儲存

Q: 如何匯出單字表？
A: 點擊 🎓 圖示 → 單字表 → 點擊「📥 匯出」
