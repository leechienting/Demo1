# 🎓 AI 語言學習助手 - Chrome 擴充功能

一個智能語言學習工具，讓你在瀏覽任何網頁時，只需選取文字就能獲得 AI 生成的解釋，並建立個人單字表。

## ✨ 功能特色

- 📖 **智能解釋**：選取任何文字，AI 會根據上下文提供詳細解釋
- 💾 **單字表管理**：一鍵儲存單字和解釋到個人單字表
- 🌐 **多語言支援**：支援繁體中文、簡體中文、英文、日文、韓文等
- 📥 **匯出功能**：可將單字表匯出為 CSV 檔案
- 🎨 **優雅介面**：現代化設計，使用體驗流暢

## 📦 安裝步驟

### 步驟 1：取得 API Key

1. 前往 [Anthropic Console](https://console.anthropic.com/)
2. 註冊或登入帳號
3. 點擊「API Keys」
4. 建立新的 API Key 並複製（注意：只會顯示一次！）

**費用說明**：
- 新用戶通常有免費額度
- 之後按使用量計費（非常便宜，一般使用約 $0.001-0.003 美元/次）

### 步驟 2：安裝擴充功能到 Chrome

1. 打開 Chrome 瀏覽器
2. 在網址列輸入：`chrome://extensions/`
3. 在右上角開啟「開發人員模式」（Developer mode）
4. 點擊左上角「載入未封裝項目」（Load unpacked）
5. 選擇這個資料夾：
   ```
   /Users/chientinglee/Documents/Project/Demo1/language-learning-extension
   ```
6. 完成！你會看到擴充功能圖示（🎓）出現在工具列

### 步驟 3：設定 API Key

1. 點擊擴充功能圖示（🎓）
2. 切換到「⚙️ 設定」頁籤
3. 在「Anthropic API Key」欄位貼上你的 API Key
4. 選擇你想要的解釋語言（預設：繁體中文）
5. 點擊「💾 儲存設定」

## 🚀 使用方法

### 查詢單字/片語

1. 瀏覽任何網頁
2. 用滑鼠選取你想了解的文字（單字、片語或句子）
3. 等待 1-2 秒，會出現 AI 解釋視窗
4. 視窗會顯示：
   - 詞性和中文意思
   - 在該上下文中的用法
   - 相關說明

### 儲存到單字表

1. 在 AI 解釋視窗中
2. 點擊「💾 加入單字表」按鈕
3. 單字會自動儲存，按鈕變成「✓ 已加入」
4. 視窗會在 1.5 秒後自動關閉

### 管理單字表

1. 點擊擴充功能圖示（🎓）
2. 在「📚 單字表」頁籤中可以：
   - 查看所有儲存的單字和解釋
   - 查看儲存時間
   - 刪除個別單字
   - 匯出為 CSV 檔案
   - 清空所有單字

### 匯出單字表

1. 在單字表頁面點擊「📥 匯出」
2. 會下載一個 CSV 檔案
3. 可用 Excel、Numbers 或 Google Sheets 開啟
4. 內容包含：單字、解釋、日期、來源網址

## 💡 使用技巧

1. **選取精準**：選取你想了解的特定單字或片語，會得到更準確的解釋
2. **包含上下文**：AI 會分析選取文字的前後文，提供更貼切的解釋
3. **學習語言**：適合學習英文、日文、韓文等外語
4. **專業術語**：也能解釋技術文件中的專有名詞
5. **定期匯出**：建議定期匯出單字表備份

## 🔧 故障排除

### 問題：選取文字後沒有反應

**解決方法**：
1. 檢查擴充功能是否已啟用（chrome://extensions/）
2. 重新載入網頁（F5 或 Cmd+R）
3. 確認已在設定中輸入 API Key

### 問題：顯示「請先在擴充功能設定中輸入 API Key」

**解決方法**：
1. 點擊擴充功能圖示
2. 進入「⚙️ 設定」
3. 輸入你的 Anthropic API Key
4. 點擊「儲存設定」

### 問題：顯示「無法取得 AI 解釋」

**可能原因**：
1. API Key 錯誤或過期
2. 網路連線問題
3. API 額度用完

**解決方法**：
1. 檢查 API Key 是否正確
2. 前往 [Anthropic Console](https://console.anthropic.com/) 檢查帳戶狀態
3. 確認網路連線正常

### 問題：解釋的語言不對

**解決方法**：
1. 進入「⚙️ 設定」
2. 在「解釋語言」下拉選單選擇正確的語言
3. 點擊「儲存設定」
