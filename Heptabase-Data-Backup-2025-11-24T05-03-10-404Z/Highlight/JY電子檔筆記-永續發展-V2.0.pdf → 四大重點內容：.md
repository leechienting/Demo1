# Highlight

上市上櫃公司永續發展**實務守則**

# Note

**四大重點內容：**

1. **落實公司治理**

   - 董事會主導永續發展。

   - 制定計畫、考量利害關係人，確保資訊即時揭露。

2. **發展永續環境**

   - 遵守環境法規與國際標準。

   - 提升能源效率、使用再生資源，推動減碳行動。

3. **維護社會公益**

   - 保護員工與消費者權益。

   - 維持供應鏈責任管理。

4. **加強永續資訊揭露**

   - 依國際準則公開資訊（如GRI）。

   - 鼓勵第三方驗證，提升資訊可信度。
