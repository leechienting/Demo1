# Highlight

永續發展目標指南

- 指引企業如何將**永續發展目標（SDGs）納入公司策略與運營中，確保能達到最佳的實施效果**，促進**綜合性、整體性和可持續性**

# Note

- **五大流程步驟**：

   1. **理解 SDGs**：

      理解每個目標的意涵與背後的重要全球議題。

   2. **定義優先順序**：

      根據企業自身的影響力與相關性，確定對應的重點目標。

   3. **制訂目標**：

      設定具體、可衡量的企業內部永續發展目標。

   4. **整合進業務流程**：

      把永續行動融入到核心業務與經營策略中。

   5. **揭露與溝通**：

      透明且具體地揭露永續成果與影響，回報給所有利害關係人。
