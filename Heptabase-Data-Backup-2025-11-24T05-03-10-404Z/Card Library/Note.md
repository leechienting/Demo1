# Note

### 文字輸入 → 透過AI → 執行多任務：

- 輸出成文字

- 把文字變成一串數字（嵌入Embedding），常用於搜尋、分群與檢索增強生成（RAG）

- 分類（把文字歸類在某一類別中），可做情感分析、垃圾郵件判斷等。

### word2vec 

- 訓練方式

   - 每個詞建立一個初始化向量

   - 隨機取一對詞，放入神經網路判斷，輸出機率值（如0.74）

      - 資料取樣通常來自「中心詞與其上下文」的視窗

   - 依照真實標註回傳誤差

   - 對應的詞向量與模型權重一起更新

- 鄰近詞相似性 

### 嵌入：就是把一個詞變成一串數字

![[../Highlight/Hands-On Large Language Models... → Untitled 2.md]]

嵌入(Embedding)是把詞表示成固定長度的向量。

圖示中的 animal、human… 是教學用的維度標籤，方便建立直覺；在真實模型裡，單一維度通常無法對應到可命名的概念。判斷兩詞是否相似，應比較整個向量（如 cosine 相似度），而非單看某一維度；各維度的數值可正可負，表示在該抽象特徵上的方向與強度。
