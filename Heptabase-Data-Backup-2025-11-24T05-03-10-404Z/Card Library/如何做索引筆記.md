# 如何做索引筆記

- 可依不同目的，為你的索引筆記套用不同框架

- 如：使用**黃金圈**框架：

   - 最內圈：為什麼 → 關注**原理** 

   - 中間圈：如何做 → 關注**方法**

   - 最外圈：做什麼 → 關注**產出跟成果**

   - 索引筆記如下：

      ![熒幕擷取畫面\_2024_12_21_21_05.png](./如何做索引筆記-assets/熒幕擷取畫面_2024_12_21_21_05.png)

- 依解決不同問題來為索引筆記套用不同的框架

   - 如年度目標設定：用SMART目標框架

   - 某重大事件的分析：用 5W1H分析框架

   - 新產品的行銷計劃：用SWOT分析框架 等等….
