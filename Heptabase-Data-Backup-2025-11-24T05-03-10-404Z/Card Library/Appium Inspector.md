# Appium Inspector

- 是一個圖形化的工具，允許測試人員在執行測試時檢查移動應用程序的 UI 元素。

- 這個工具的主要目的是幫助使用者了解應用程序的結構、元素屬性，並生成相應的定位 (locators) 來進行自動化測試。

- 啟動Appium inspector前，記得先啟動Appium Server。

- 每次的連線都算是一個 Session。
