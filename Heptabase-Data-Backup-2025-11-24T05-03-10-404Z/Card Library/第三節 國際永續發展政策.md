# 第三節 國際永續發展政策

![[../Highlight/JY電子檔筆記-永續發展-V2.0 (1) → Untitled 7.md]]

***++整理這段內容，用條列方式，不要遺漏資訊++*** 

- **名稱：** 政府間氣候變遷專門委員會 (IPCC) 

- **成立目的與沿革：** 

   - 1972年：聯合國首次舉辦人類環境會議，通過《斯德哥爾摩宣言》，並決議設置聯合國環境規劃署 (UNEP)。 

   - 1988年：由世界氣象組織 (WMO) 與 UNEP 共同成立 IPCC。 

- **組織性質：** 隸屬於聯合國下的獨立組織。 

- **設立目的：** 

   - 從科學、技術、社會經濟觀點，針對人為起源的氣候變化、影響、適應以及緩和對策進行綜合性評價。

   - 主要負責研究與評估由人類活動所造成的氣候變遷現象與問題。

   - 提供氣候變遷相關的科學評估報告。

***++是否有AR1～AR4評估報告？若有，其內容為何？用一句話來描述++*** 

是的，除了您提到的第五次 (AR5) 和第六次 (AR6) 評估報告之外，IPCC 也發布了 AR1 到 AR4 的評估報告。IPCC 自 1988 年成立以來，已發布了六次綜合評估報告，這些報告是全球最全面的氣候變遷科學報告 。

以下是 AR1 至 AR4 評估報告的重點概述：

- **第一次評估報告 (AR1) (1990年)**：報告強調氣候變遷作為一個具有全球影響的挑戰，需要國際合作應對，並在聯合國氣候變遷綱要公約 (UNFCCC) 的制定中發揮了關鍵作用 。

- **第二次評估報告 (AR2) (1995年)**：報告提供了重要的科學和社會經濟資訊，表明「證據平衡顯示人類活動對全球氣候有可辨識的影響」，為《京都議定書》的通過奠定了基礎 。

- **第三次評估報告 (AR3) (2001年)**：報告將重點放在氣候變遷的影響以及適應的必要性上，並提供了更強而有力的證據，證實地球正在暖化且人類活動是主要原因 。

- **第四次評估報告 (AR4) (2007年)**：報告為後京都協議奠定了基礎，明確指出「氣候系統的暖化是毋庸置疑的」，並且「**非常可能**」是人類活動造成的溫室氣體增加所致 。

***++請說明UNFCCC是在什麼歷史背景下提出的？\

UNFCCC（聯合國氣候變遷綱要公約）的相關說明條列如下：

- 歷史背景

   - 1988年IPCC成立，其1990年報告促成氣候變遷公約談判。

   - 1990年聯合國大會決議設立INC。

   - 1992年5月9日UNFCCC文本於紐約聯合國總部通過。

   - 同年6月，UNFCCC與其他姊妹公約在里約地球高峰會開放簽署。

   - UNFCCC於1994年3月21日正式生效，目前有198個締約方。

- 原因

   - UNFCCC最終目標是穩定大氣中溫室氣體濃度，防止危險人為干擾。

   - 此目標旨在使生態系統自然適應、確保糧食生產及永續經濟發展。

   - 公約強調工業化國家應率先減排，並為開發中國家提供資金技術支持。

- 目前進展

   - UNFCCC工作重點是實施2016年生效的《巴黎協定》，目標限制全球溫升在1\.5°C。

   - 《巴黎協定》要求每五年進行一次「全球盤點」，評估各國氣候目標進展，首次已於2023年COP28完成。

   - 各國需定期提交並強化國家自定貢獻（NDCs），COP30前應提交2035年的NDC。

***++請舉例，主要不利衝擊是什麼？++***

根據您提供的資料，雖然文件中沒有直接舉出「主要不利衝擊」（Principal Adverse Impact, PAI）的具體案例，但我們可以從內容推斷其意涵。

「主要不利衝擊」指的是投資決策對永續性因素（如環境、社會和員工事務、人權、反腐敗和反賄賂事項）造成的負面影響。[\[1\]\[2\]](https://app.heptabase.com/f3c4fc0d-fe07-4b10-932c-6f968c1f5ab0/card/341c60c3-9796-4118-868a-8856f1feef77?pdfPage=19&pdfBoundingBox=%7B%22left%22%3A32.725%2C%22top%22%3A452.458%2C%22width%22%3A525.9799999999999%2C%22height%22%3A87.464%7D)

文件中提到，SFDR 的監管技術標準會對「主要不利影響指標」提出具體的計算方法和修改建議，其中包含額外的「社會指標」。[\[3\]](https://app.heptabase.com/f3c4fc0d-fe07-4b10-932c-6f968c1f5ab0/card/341c60c3-9796-4118-868a-8856f1feef77?pdfPage=19&pdfBoundingBox=%7B%22left%22%3A32.725%2C%22top%22%3A668.595%2C%22width%22%3A527.765%2C%22height%22%3A106.80700000000002%7D) 這表示主管機關會定義一系列明確的指標來衡量這些衝擊。

**舉例來說，可能的「主要不利衝擊」指標可能包括：**

- **環境方面**：

   - 溫室氣體排放量

   - 對生物多樣性造成負面影響的活動

   - 水資源消耗與污染

   - 廢棄物產生量

- **社會方面**：

   - 違反國際勞工標準或人權公約

   - 性別薪酬差距

   - 工作場所意外事故率

因此，如果一項投資流向一家溫室氣體排放量超標、或存在剝削勞工行為的企業，就可以被視為對永續性造成了「主要不利衝-擊」。

***++所謂金融市場參與者、和財務顧問具體是指誰？++***

根據永續金融揭露規範（SFDR），「金融市場參與者」主要指的是以下類型的機構：

- 投資公司

- 基金公司

- 保險公司

- 資產管理業者

- 投資基金業者

這些實體必須公開揭露其所投資企業在環境、社會與公司治理（ESG）方面的落實狀況，並負有加強 ESG 相關資訊強制揭露的義務，例如協定前揭露、階段報告揭露以及永續投資決策中的重大不利影響揭露等。

「財務顧問」則是指在 SFDR 規範下，需要與金融市場參與者一同向最終投資者提供金融產品資訊，並受監管的實體。雖然內文沒有提供具體範例，但他們和金融市場參與者一樣，都需揭露促進環境或社會特徵的產品，以及促進永續性目標的產品相關資訊。

好的，我來用白話文說明這三者之間的關係：

首先，你可以把這三者想像成一套「要求企業揭露永續資訊」的系統：

- **CSRD (企業永續報告指令)**：這可以看作是歐盟政府發布的「**一道命令**」或「**一份法律**」。這道命令要求特定的上市公司和大型企業，必須公開它們在環境和社會方面的表現、面臨的風險，以及它們的活動對環境和社會造成的影響。簡單來說，它規定了「誰要報告」和「報告什麼樣的大方向」。

- **EFRAG (歐洲財務報導諮詢小組)**：這是一個「**專家小組**」或「**智囊團**」。它的任務就是根據CSRD這道命令的精神，去實際「**制定**」一套更詳細、更具體的報告規則和標準，讓企業知道報告時要遵守哪些細節。

- **ESRS (歐洲永續發展報告準則)**：這就是由EFRAG這個專家小組實際「**制定出來的具體報告規則**」或「**標準**」。當企業收到CSRD的命令後，它們就會參考ESRS來準備自己的永續報告。這些準則會告訴企業，在環境（例如碳排放、水資源使用）和社會（例如員工權益、供應鏈人權）方面，具體要揭露哪些資訊、怎麼揭露。

總結來說，它們的關係可以這樣表示：

***++歐盟政府發布 CSRD (命令)++*** 

***++↓++*** 

***++EFRAG (專家小組) 制定++*** 

***++↓++*** 

***++ESRS (具體報告規則)++***

也就是說： ++歐盟透過 **CSRD**，**要求** 企業進行永續報告。 **EFRAG** 這個組織，**負責制定** 報告的具體標準。 而 **ESRS**，就是 **EFRAG 制定出來的那些具體標準**，企業就是依照這些標準來撰寫永續報告++。
