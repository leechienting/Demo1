1. 關於寫作

   1. 一個好的系統(**卡片盒筆記法**) P47

      1. 必須是你**可以信任**的系統。

      2. 能夠讓**流程順利進行。**

      3. 可以不斷地**增加、組織訊息，並闡述新的見解。**

   2. 『清晰的寫作結構』與『訂定寫作計劃』的差異 P48

   3. 寫作不是一種線性的過程，我們必須一直在不同的工作項目中切換。

   4. 一則筆記因為有了他自己的脈絡，才有了它的價值。

   5. 當我們把工作的方式設定得太僵硬，而沒有彈性的時候，我們就會被卡住，看著情況朝自己無法控制的方向發展。要保持掌控感的最好方法就是讓自己持續掌控一切不要把自己局限在最開頭發想的那個想法。 P66

   6. 成功並不是靠著**強大的意志力**跟**克服抗拒**才達成的，而是因為有設計卓越的工作環境（日常的工作流程），可以在開頭就避免產生抗拒(避開阻力)。P67

      - 這是魯曼跟他的卡片盒一起工作的方式，他因此得以自由與靈活的在不同的**工作項目**和**思考層次**中不斷切換。

2. 四大基本原則

   1. 寫下來是唯一重要的事

      1. 專注於寫作上，會改變你閱讀的方式：你會更專注在最相關的領域上，因為你知道無法把所有東西都記下來，你會投入更多心力在閱讀上，因為如果你沒有讀懂內容，是無法用自己的話來重新闡釋其含義，使你更有可能牢牢記住。你還得對於獨到的東西更進一步的思考，因為你必須把它們轉換成新的東西。P107

      2. 刻意練習是唯一真正讓我可以更精進自己的方法。

   2. 簡單最重要

      1. 統一的卡片格式：正因為有統一的格式，所以這些筆記才能夠建立起群聚效應。而統一格式也是觸發思考跟寫作的關鍵所在，因為規格混亂會產生不必要的複雜狀況或決定。

      2. 每條筆記都是相同的格式，放在相同的地方，因此他們日後才能組合並銜接，成為新的東西，而且沒有一個想法會因為不知放哪裡或標記在哪裡而被捨棄。

   3. 沒有人是從零開始

      1. 寫作不應從零開始做線性的發展，而是循環的過程。

      2. 我們寫作時不可能是從自己那不太可靠的頭腦裡抓取資料，盲目擬定計劃，然後依據這種計劃行動；我們的計劃是由於興趣、好奇心、直覺所引導的。

      3. 透過實際的閱讀、思考、討論、書寫、發展想法，然後不斷增長，最後形成寫作計劃。

      4. 只要專注在自己有興趣的領域，持續將自己動腦思考的過程記錄下來，那麼主題、問題和論述不需要費力，自然會從素材中浮現。

      5. 與其把時間花在擔心找不到適合的題目來寫，不如把時間用在好好去了解自己已經感興趣的主題上，並且為了正確決定要寫哪個題目而好好去閱讀、思考、寫筆記。

   4. 讓工作帶領你往前

      1. **好的工作流程**能夠輕易轉變為一種正向的循環：正向的經驗激勵我們放輕輕鬆鬆地去做接下來的任務，因此幫助我們在所做的事情上表現得更好，然後回過頭來，我們更有可能樂在工作中。

      2. 擁有成長型心態當然重要，但這還不夠。同樣重要的是要有一個『能讓反饋以切合實際的方式產生循環』的學習系統。

      3. 採用循環性的方式而非線性的方式，可以讓你進行多次**反饋循環**，使你有機會一邊進行一邊改善。這樣不僅增加學習的機會，還有機會可以修正我們無可避免一定會犯的錯誤。相較於工作結束後一次出現大規模的反饋，反饋循環當中的反饋多半比較小，較易被接受。

      4. 用自己的話來寫出自己理解的內容，這種能力是每個寫作者都需要具備的基本技能，才可能體悟到自己是否真正了解。

      5. 永久筆記可說是擁有一種內建的反饋循環：用文字表達自己的想法，可以讓我們明白自己有沒有真的思考得很透徹。當我們想要把先前寫下來的筆記做連結時，這個系統會好含糊的向我們顯現出其中的矛盾、不一致、重複之處。

      6. 卡片盒的目的不是要收集筆記，也不是讓你可以回頭找到特定的筆記，而是讓你看見有關聯的事實，並且讓各種想法教會融合，使你產生真正的見解。

3. 六個寫作成功的步驟

   1. 各自分開與相互串連的工作項目

      1. 人們無法同時將專注力放在好幾件事情上。

      2. 不要做計劃，要做專家。

      3. 不停從經驗中學習，不要有太多的事前規劃。 

      4. 經由犯錯中吸取經驗，慢慢內化為自己的知識，成為專家。

      5. 將大腦的**短期記憶**用卡片盒筆記記下來，清出空間來讓大腦做更有用的事情；當短期記憶中的事情清除了，就更能有專注力處理更重要的事情。

      6. 使用卡片盒筆記，因為寫作格式、工作流程固定， **減少做決定的次數**（減少決定文章架構、知識重要性、連結性）。
