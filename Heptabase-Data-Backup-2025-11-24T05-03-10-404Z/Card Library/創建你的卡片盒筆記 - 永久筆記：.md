# 創建你的卡片盒筆記 - 永久筆記：

- 從五顏六色的**文獻筆記**中抽取特定顏色的積木， 變成**永久筆記**。

- 未來的任何時刻我們都可以去拿這個特定顏色的積木來使用（**永久筆記**），而不需要回頭去重複看五顏六色的積木（**文獻筆記**）。

- 要從**靈感筆記或文獻筆記轉成永久筆記**時，記得重建一張新的永久筆記卡片，保留原始的靈感筆記或文獻筆記。

- 要能夠**完整陳述一個脈絡**。

   - 也就是用自己的話，重新解釋的過程， 這有助於加深我們對於這個資訊的理解。

   - 他涉及到了認知的深層處理， 轉化為自己的語言跟想法， 可以幫助我們加深記憶，並且提取資訊。

   - 我們要追求的是， 經過這個闡示的過程， 將資訊理解得更深刻。

   - 不要求先記得，要先求理解； 當你理解了，會更容易記得。

   - 跟已有的知識跟經驗相互連結。

   - 檢查自己的理解是否足夠正確與完整。

   - 不只是收集想法，而是發展想法。

- 我們如何**建立內部連結（永久筆記間的連結）**：

   - 我們需要把新的筆記與舊的筆記之間建立起筆記連結。

   - 脈絡相互連結的筆記，更容易在未來派上用場。

   - 我們要思考：**未來如何想起這則筆記？**

   ### [怎麼建立卡片連結 (我們可以思考的方向有哪些)：.md](./怎麼建立卡片連結%20\(我們可以思考的方向有哪些\)：.md)

- 標題命名：

   - 說明：

      - 一開始先給隨便一個標題，等我們**闡述**和**連結**完後，再來重新命名標題。

      - 標題代表這則筆記的摘要，且一看標題，就能知道這則筆記的內容在說什麼。

      - 未來再重複引用這則永久筆記時，只要透過標題，就知道這則筆記是否是自己需要，大大提高卡片盒的效用。

   - 永久筆記的標題，可以是：

      - 描述一個事實或現象

      - 闡述一個觀點或想法

   - 命名原則：

      - 準確的標題：只需理解標題即可，而不需要去知道細節內容，增加使用筆記效率。

         → 用完整語句(一句話)下標題：若無法用一句話下標題，則需思考是否筆記內容涵蓋太多的概念，是否需要拆分筆記內容。

         標題是**概念導向**，標題中的文字，不一定要從內文來。

      - 正向的框架：

         - 要用**正面**的語句描述標題。

      - 未來的可發現性

         - 標題要很容易被找到，一目了然。

         - 想像未來的你，當需要用到這則筆記時，要怎麼樣地很快發現、找出這則筆記。

      - **避免**使用清單文做標題：

         - 如：**時間管理的10個好習慣**（可能屬於10個脈絡或知識點）

         - 因為當你看完標題後，還是不知道內容是什麼？除非你進去看文章的內容，這是屬於**策展性質**的筆記。
