# 藍玉案

藍玉案是明朝初期，明太祖朱元璋為鞏固皇權、清除開國功臣而發動的一場大規模政治肅清事件，與[胡惟庸案.md](./胡惟庸案.md)合稱「胡藍黨獄」，對明初政治格局產生深遠影響。[\[1\]](https://app.heptabase.com/f3c4fc0d-fe07-4b10-932c-6f968c1f5ab0/card/b1b1ccfa-90b4-4349-800f-8620243e76fe#1a4594bb-b8fc-4f0d-aa71-afbdd2db54ff)

**基本資訊**

- **發生時間：** 明洪武二十六年（1393年）。

- **地點：** 主要發生於明朝京師應天府（今南京）。

- **歷經皇帝：** [(1)明太祖 朱元璋.md](./\(1\)明太祖%20朱元璋.md)。[\[1\]](https://app.heptabase.com/f3c4fc0d-fe07-4b10-932c-6f968c1f5ab0/card/b1b1ccfa-90b4-4349-800f-8620243e76fe#1a4594bb-b8fc-4f0d-aa71-afbdd2db54ff)

**背景原因**

- **藍玉功高震主，行為驕橫跋扈：** 藍玉身為開國大將，戰功赫赫，尤其在捕魚兒海大捷後被封涼國公。但他自恃功高，常擅自升降將校、不聽君命，甚至毀關強入、霸佔民田、私販食鹽，並有強姦元妃等不法行為。

- **朱元璋猜忌功臣：** 明太祖朱元璋對功臣始終抱有強烈猜忌，恐其威脅朱家江山。

- **太子朱標早逝與儲君危機：** 洪武二十五年（1392年）太子[朱標（懿文太子）.md](./朱標（懿文太子）.md)去世，繼位的皇太孫[(2) 建文帝 朱允炆.md](./\(2\)%20建文帝%20朱允炆.md)年幼，朱元璋擔心藍玉等武將集團難以駕馭，故決定清除潛在威脅，為皇太孫鋪路。

**事件過程**

- [錦衣衛.md](./錦衣衛.md)**：** 洪武二十六年（1393年）二月初八，錦衣衛指揮蔣𤩽告發藍玉謀反。

- **迅速逮捕處決：** 藍玉隨即被捕，審訊後被處以凌遲之刑，並株連三族。

- **大肆株連清洗：** 朱元璋頒布《逆臣錄》，列名者有一公、十三侯、二伯等，連同其黨羽，牽連被殺者高達數萬人。

**重要人物**

- [(1)明太祖 朱元璋.md](./\(1\)明太祖%20朱元璋.md)**：** 明朝開國皇帝，主導藍玉案的發起者與執行者，旨在清除功臣以鞏固皇權。[\[1\]](https://app.heptabase.com/f3c4fc0d-fe07-4b10-932c-6f968c1f5ab0/card/b1b1ccfa-90b4-4349-800f-8620243e76fe#1a4594bb-b8fc-4f0d-aa71-afbdd2db54ff)

- **藍玉：** 明朝涼國公、大將軍，捕魚兒海大捷的功臣，因其驕橫與朱元璋的猜忌而被羅織謀反罪名處決。[\[1\]](https://app.heptabase.com/f3c4fc0d-fe07-4b10-932c-6f968c1f5ab0/card/b1b1ccfa-90b4-4349-800f-8620243e76fe#1a4594bb-b8fc-4f0d-aa71-afbdd2db54ff)

- [朱標（懿文太子）.md](./朱標（懿文太子）.md)**：** 皇太子，藍玉曾是其太子太傅，他的早逝是促使朱元璋決定清除功臣的重要因素。

- [(2) 建文帝 朱允炆.md](./\(2\)%20建文帝%20朱允炆.md)**：** 皇太孫，朱元璋為其未來繼位掃清障礙。

- **蔣𤩽：** 錦衣衛指揮，告發藍玉謀反。

**結果** 藍玉及其黨羽被族誅，大量開國功臣、文臣武將遭到殺戮，共計約一萬五千人，甚至數萬人。藍玉的皮被剝下，懸掛城門示眾。[\[1\]](https://app.heptabase.com/f3c4fc0d-fe07-4b10-932c-6f968c1f5ab0/card/b1b1ccfa-90b4-4349-800f-8620243e76fe#1a4594bb-b8fc-4f0d-aa71-afbdd2db54ff)

**歷史影響**

- **皇權高度集中：** 徹底清除了對皇權構成潛在威脅的功臣集團，使明朝皇權達到前所未有的高度集中。

- **文武人才凋零：** 大批有軍事才能和治國經驗的文臣武將被殺，導致明初政局不安，尤其在「[靖難之役.md](./靖難之役.md)」時，建文帝（朱允炆）身邊無將可用，間接加速了朱棣奪權成功。

- **專制統治加劇：** 胡藍黨獄標誌著朱元璋以極端殘酷手段維護皇權，加劇了明朝的專制統治色彩。[\[1\]](https://app.heptabase.com/f3c4fc0d-fe07-4b10-932c-6f968c1f5ab0/card/b1b1ccfa-90b4-4349-800f-8620243e76fe#1a4594bb-b8fc-4f0d-aa71-afbdd2db54ff)

**關鍵時間線**

- **1388年：** 藍玉大破北元軍於捕魚兒海，晉封涼國公。

- **1392年：** 太子朱標去世。

- **1393年2月：** 錦衣衛告發藍玉謀反，藍玉被捕處決。

- **1393年5月：** 朱元璋頒布《逆臣錄》，大肆株連清洗。
