# 風格類

### **藝術風格類**

| **藝術風格提示詞** | **說明** | 
|---|---|
| 油畫風 | 像油畫一樣厚重的筆觸與色彩，適合復古、莊嚴主題 | 
| 水彩畫風 | 顏色暈染、柔和，帶有透明感與夢幻效果 | 
| 鉛筆素描風 | 黑白線條構圖，強調陰影與手繪感 | 
| 墨筆畫風 / 水墨畫風 | 類似中國水墨畫，黑白為主，有筆觸與留白感 | 
| 木炭畫風 | 粗曠灰階風格，有強烈的陰影與輪廓 | 
| 畢加索 | 線條奇怪但有特色，藝術感很強 | 
| 塗鴉風格 | 自由、隨性，像街頭牆上的塗鴉一樣，有種年輕活力的感覺 | 
| 波西米亞風格 | 顏色很多、圖案很多，看起來自由又浪漫 | 
| 蒙德里安 | 畫面很漂亮、很華麗，人物柔美，像老海報那種感覺 | 
| 非洲未來主義 | 把非洲文化跟科技感結合起來，看起來很酷又有創意 | 
| 義大利巴洛克風 | 光影強烈，古典藝術感 | 
| 法國印象派 | 筆觸自由、用色夢幻 | 
| 德國表現主義 | 誇張變形、情緒強列 | 
| 蘇聯構成主義 | 幾何圖形、極簡有力感 | 

### **攝影風格類**

| **攝影風格提示詞** | **說明** | 
|---|---|
| 照片寫實風 / 寫實攝影風 | 圖像看起來就像真實拍攝的照片 | 
| 電影光影風 | 像電影畫面一樣的光影效果，有戲劇感 | 
| 微距攝影風 | 極近距離的拍攝視角，突出細節 | 
| 人像攝影風 | 聚焦人物臉部與情感表情，適合肖像主題 | 
| 復古照片風 | 加入懷舊濾鏡、泛黃、膠片質感等效果 |

### **數位與幻想風格類**

| **數位畫風提示詞** | **說明** | 
|---|---|
| 數位插畫風 | 銳利線條與鮮明色彩，適合現代風格畫面 | 
| 賽博龐克風 | 帶有未來感與科技感，常見霓虹、都市夜景 | 
| 奇幻風 / 魔幻風 | 帶有魔法、神話角色、幻想世界的風格 | 
| 動漫風 / 日本動畫風 | 類似日本動畫或漫畫角色的繪畫方式 | 
| 像素風 / 像素藝術風 | 類似早期電玩遊戲風格，由像素組成的圖像 | 
| 美式漫畫 | 像超級英雄漫畫那樣，人物很有特色，動作誇張又有戲劇感 | 
| 包豪斯 | 看起來很簡潔、實用又有設計感，像現代建築或家具的感覺 | 
| 吉卜力風格 | 有點像宮崎駿的動畫，溫暖又帶點奇幻的感覺 | 
| 3D卡通 | 有立體感，看起來像動畫角色一樣會動、很生動 | 
| ASCII art | 用英文字母或符號排成的圖，很像以前電腦螢幕上的藝術圖 | 
| 剪影 | 只看物體的外型輪廓，就像影子那樣，簡單又有形 | 
| 少女漫畫風 | 細膩筆觸、大眼睛、唯美 | 
| 少年漫畫風 | 動作誇張、熱血 | 
| 青年漫畫風 | 寫實比例、劇情沉穩 | 
| 寫實插畫風 | 接近照片、光影細膩 | 
| 法式漫畫風  | 水彩感柔和筆觸、文藝氣氛 |

### **可愛與童話風格**

| **童話風格提示詞** | **說明** | 
|---|---|
| 可愛風 / 卡哇伊風 | 色彩鮮明、線條圓潤，表情誇張，充滿萌感 | 
| Q版風格 | 人物比例誇張（頭大身小），可愛有趣 | 
| 夢幻風 / 粉色夢幻風 | 色彩柔和、光暈特效，像童話世界的氛圍 | 
| 童話繪本風 | 適合兒童故事，色彩溫馨、構圖簡單 |

### **東方風格類**

| **東方風格提示詞** | **說明** | 
|---|---|
| 浮世繪風 / 日本浮世繪 | 日本傳統木版畫風格，古典與色塊構圖 |
