# **什麼是學習**？

- 就是把新的知識，縫接到舊的知識上面，**連接知識**就是我們深入學習某件事情的關鍵。很多人可能學了很多知識，但忘了把大部分的知識連接在一起，所以他們的學習就還不夠完整。

- 淺層的學習 → 專注在輸入

   - 不斷地在新知識上疊加薪資，忘了跟舊知識做整合。

- 深層的學習 → 專注在輸出

   - 把新知識縫接到原有的知識結構中，並用自己的話重新詮釋。

- 我們必須把自己已知的事物當作基準點，才有辦法去理解、記憶跟創造我們還沒有理解的未知事物。

   - 如用二個舊觀念，去創造一個新觀念

      - 多巴胺反應（舊知識）

      - 蕃茄鐘工作法 （舊知識）

      - 產出新知識
