# 提示詞範本-進階版

#### ➤ 進階版提示詞（適合深化情感層次）

請你幫我撰寫一篇心得文，並遵循以下指引：

1. 文章開頭，請用一個與台灣日常生活相關的小故事或畫面（如：雨夜的台北捷運、淡水老街邊散步、喝熱豆漿的早晨）引入主題。

2. 在正文部分，請加入一到兩個具體感官描寫（如氣味、聲音、觸感），讓讀者更能代入。

3. 表達個人觀點時，請保持自然真誠，避免使用過於高大上的抽象詞（如：宇宙能量、生命意義），而是用具體經驗說明。

4. 最後一段結論，請用一句具有餘韻的短句收尾，例如台灣俗語（如：「吃虧就是佔便宜」、「一步一腳印」）或簡單有力的自我期許。
