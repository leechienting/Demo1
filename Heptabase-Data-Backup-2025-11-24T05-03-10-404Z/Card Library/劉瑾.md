# 劉瑾

### 歷史定位

明朝正德年間權傾一時的「立皇帝」，透過掌控東廠與西廠，建立起明朝歷史上極為專橫的宦官專權體制，是導致明武宗荒廢朝政與大明國力轉衰的關鍵人物。

### 基本資訊

- **生卒年份**：1451年－1510年（享年 59 歲）

- **籍貫**：陝西興平

- **主要身份**：司禮監掌印太監、提督東廠、西廠及內行廠

- **權力代稱**：當時民間稱其為「立皇帝」，意指站著的皇帝（對應坐著的皇帝明武宗）。

### 核心成就（專權事蹟）

1. **建立特務統治網絡**：除了既有的[東廠.md](./東廠.md)、[西廠.md](./西廠.md)，劉瑾另外創設了「[內行廠.md](./內行廠.md)」，不僅監察臣民，甚至連東西廠的特務也在其監控之下，將恐怖統治推向極致。

2. **推動嚴酷的新政改革**：以整頓朝綱為名，實行嚴苛的罰款與降級制度，大量驅逐正直官員（如[王守仁（王陽明）.md](./王守仁（王陽明）.md)），並實施「罰米例」搜刮官員財富，充實私人金庫。

3. **架空皇權與內閣**：利用明武宗朱厚照貪玩的心理，引導皇帝沉迷於豹房享樂，自己則代理批閱奏章（批紅），實質上掌握了國家的最高決策權。

4. **組建「八虎」政治集團**：以劉瑾為首，集結馬永成、高鳳等八名宦官，形成緊密的利益共同體，徹底把持朝政並排除異己。

### 重要關聯人物

- [(11) 明武宗 朱厚照.md](./\(11\)%20明武宗%20朱厚照.md)：劉瑾的權力來源。劉瑾自幼服侍太子朱厚照，深知其喜好，透過進獻鷹犬歌舞博取信任，是武宗最依賴的「伴侶」與代理人。

- **楊一清**：時任三邊總制，劉瑾的終結者。他在平定安化王之亂後，利用獻俘的機會，與太監張永密謀，最終促成劉瑾倒台。

- **張永**：「八虎」之一，但與劉瑾有隙。他在楊一清的策動下，趁武宗酒醉時揭發劉瑾謀反，是執行逮捕行動的關鍵人物。

- [王守仁（王陽明）.md](./王守仁（王陽明）.md)：著名受害者。因上書彈劾劉瑾，遭到廷杖並貶謫至貴州龍場，這場政治迫害反倒成為他「龍場悟道」的契機。

### 歷史影響

劉瑾的專權雖然僅維持了約++五年++，但對明朝中後期的政治生態造成了不可逆的破壞。首先，他開創了宦官高度干預朝政、甚至凌駕於司法之上的惡例，使得「廠衛」機構徹底成為政治鬥爭的工具。其次，他的斂財行為極度瘋狂（據傳抄家時黃金白銀堆積如山），加劇了官場貪腐與社會貧富差距，直接導致了當時的地方叛亂（如安化王之亂、劉六劉七起義）。最後，正是因為他的迫害，間接促成了[王守仁（王陽明）.md](./王守仁（王陽明）.md)心學的誕生與傳播，這算是他留給歷史一個意外的反面遺產。

### 過世情況

**時間與死因**：1510年（正德五年），因謀反罪被處以「凌遲」極刑。 **相關爭議與細節**：劉瑾的死狀極為淒慘，依律被判凌遲3357刀，分三天執行。據記載，第一天受刑後他甚至還喝了兩碗粥。行刑完畢後，許多曾受其害的人家紛紛出錢買下他身上的肉生吃，以洩心頭之恨。他的倒台標誌著正德年間第一階段宦官亂政的結束，但也預示著明朝內部權力結構已嚴重失衡。

### 附錄：關鍵年份時間線

- **1505年（弘治十八年/正德元年）**：明孝宗駕崩，太子[(11) 明武宗 朱厚照.md](./\(11\)%20明武宗%20朱厚照.md)即位（明武宗）。劉瑾升任司禮監鐘鼓司太監，開始受寵並掌握權力。

- **1506年（正德元年）**：內閣大臣劉健、謝遷等人聯名彈劾「八虎」，失敗後被迫致仕。劉瑾正式掌權，升任司禮監掌印太監。

- **1508年（正德三年）**：劉瑾設立「[內行廠.md](./內行廠.md)」，並創立嚴苛的「罰米例」，權力達到頂峰，文武百官甚至需向其跪拜。

- **1510年（正德五年）**：安化王朱寘鐇以「清君側、誅劉瑾」為名起兵。亂平後，楊一清與張永聯手揭發劉瑾謀反。

- **1510年（正德五年八月）**：劉瑾被捕，並於當月在北京被處以凌遲處死，結束其傳奇而罪惡的一生。
