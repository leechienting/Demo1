# 劉基（劉伯溫）

劉伯溫是元末明初的軍事家、政治家與文學家，他以卓越的謀略輔佐朱元璋建立明朝，是明朝的開國元勳，被譽為「吾之子房」。

**基本資訊**

- **生卒**：1311年7月1日－1375年5月16日（63歲）

- **主要身份**：軍事家、政治家、文學家

- **爵位**：誠意伯

- **諡號**：文成

**核心成就**

- **奠定帝業**：向朱元璋提出「先滅[陳友諒 (漢政權).md](./陳友諒%20\(漢政權\).md)，再滅[張士誠 (大周政權).md](./張士誠%20\(大周政權\).md)，而後北伐中原」的策略方針，為明朝的建立擘劃了清晰的藍圖。

- **運籌帷幄**：在與陳友諒的[鄱陽湖大戰.md](./鄱陽湖大戰.md)中，獻策移軍湖口、設伏取勝，扭轉了戰局，為朱元璋掃除最強大的敵人。

- **制度創建**：參與制定明朝的軍衛法與《大明律》，整肅綱紀，並呈上《戊申大統曆》，為國家典章制度的建立做出重要貢獻。

- **文學思想**：著有《郁離子》等作品，透過寓言故事針砭時弊，展現其政治思想與文學才華，與[宋濂.md](./宋濂.md)、高啟並稱「明初詩文三大家」。

**重要人物**

- [(1)明太祖 朱元璋.md](./\(1\)明太祖%20朱元璋.md)：賞識劉伯溫的才智，視其為首席謀臣，常與之密議軍國大事。

- [胡惟庸.md](./胡惟庸.md)：與劉伯溫有政治鬥爭，後世多認為劉伯溫的死與胡惟庸的毒害有關。

- [李善長.md](./李善長.md)：同為開國功臣，但與劉伯溫存在政治矛盾。

**歷史影響** 劉伯溫不僅是傑出的開國功臣，其神機妙算的形象更在民間廣為流傳，被後人比作諸葛亮，成為智慧的化身。 他的傳說故事與預言（如《燒餅歌》）在民間文學中佔有一席之地，深刻影響了後世文化。 此外，他臨終前預言劉家將在第九代後再度興盛，百年後其九世孫劉瑜獲朝廷重用，也為他的傳奇增添了神秘色彩。

**過世原因** 洪武八年（1375年），劉伯溫因病告老還鄉。 在此之前，他曾因風寒由丞相[胡惟庸.md](./胡惟庸.md)派御醫前來診治，服藥後病情加劇，腹中疑似有硬塊。 返鄉後一個月即病逝。 根據《明史》及後續揭露的資料，主流觀點認為他是被胡惟庸下毒害死，但也有人懷疑此事的背後或許有朱元璋的默許。

**關鍵年份時間線**

- **1333年**：考中進士，開始其仕途生涯。

- **1360年**：應朱元璋之邀出山，成為其重要謀臣。

- **1363年**：在[鄱陽湖大戰.md](./鄱陽湖大戰.md)中獻策，助朱元璋大敗陳友諒。

- **1370年**：因開國之功，受封為誠意伯。

- **1375年**：因病逝世，享年63歲。
