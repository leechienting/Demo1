# 提示詞

請為\[人物姓名\]撰寫一份精簡的生平簡介，用於Heptabase知識卡片。

要求：

1. 總字數控制在500-600字

2. 採用以下結構：

   - 開頭：一句話概括此人的歷史定位（是誰、為何重要）

   - 基本資訊：生卒年（歲數）、主要身份（若此人為皇帝，加上廟號、年號、在位多少年、與上任皇帝的關係）

   - 核心成就：3-4個最重要的事蹟或貢獻（用條列式）

   - 重要人物：列出其身邊重要人物

   - 歷史影響：說明對後世的影響

   - 過世原因：說明過世原因

3. 寫作原則：

   - 使用簡潔有力的語言

   - 突出最具代表性的事蹟

   - 避免過多細節

   - 讓完全不認識此人的讀者能快速掌握重點

4. 可選附加：關鍵年份時間線（3-5個關鍵時間點）
