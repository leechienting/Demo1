### 🛠️ 工具腳本

#### 1. transcribe_audio.py
**用途**：使用 OpenAI Whisper 轉錄音訊檔案（標準版）

**使用方法**：
```bash
python3 transcribe_audio.py
```

**設定**：
- 模型：base（平衡速度與準確度）
- 語言：中文 (zh)
- 可修改 `model_size` 為 'tiny', 'small', 'medium', 'large'

#### 2. transcribe_audio_fast.py
**用途**：快速版本轉錄（使用 tiny 模型）

**使用方法**：
```bash
python3 transcribe_audio_fast.py
```

**特點**：
- 速度快 3-4 倍
- 準確度稍低（約 85-90%）
- 適合快速預覽

#### 3. analyze_speakers.py
**用途**：自動分析並標註發言人

**使用方法**：
```bash
python3 analyze_speakers.py
```

**功能**：
- 識別發言人角色（技術人員、主管、參與者）
- 合併連續相同發言人的片段
- 生成發言統計
